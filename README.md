# Otus-RL-homework

Домашние задания по Reinforcement Learning для курса Otus.

## Домашнее задание 1: Q-Learning для Taxi-v3

Реализация Q-Learning агента для решения задачи Taxi-v3 из библиотеки Gymnasium.

### Описание задачи

В задаче Taxi-v3 такси должно:

1. Забрать пассажира в одном из 4 обозначенных мест (R, G, Y, B)
2. Доставить пассажира в пункт назначения

**Система наград:**

- +20 за успешную доставку пассажира
- -1 за каждый шаг (стимулирует быстрое решение)
- -10 за неправильные действия

**Пространство:** 500 состояний, 6 действий

### Файлы домашнего задания

- [notebooks/taxi-v3.ipynb](notebooks/taxi-v3.ipynb) - основной notebook с реализацией
- `trained_agent.pkl` - сохраненный обученный агент (создается после обучения)
- `training_results.png` - графики обучения (создается после обучения)

### Запуск

```bash
uv run jupyter notebook notebooks/taxi-v3.ipynb
```

Запустите все ячейки по порядку. Обучение займет несколько минут.

### Результаты

После обучения агент должен достичь средней награды 7-9 за последние 100 эпизодов.

## Установка

Проект использует [uv](https://github.com/astral-sh/uv) для управления пакетами Python.

### Требования

- Python 3.13+
- Менеджер пакетов uv

### Инструкция по установке

1. Клонировать репозиторий:
```bash
git clone <repository-url>
cd Otus-RL-homework
```

2. Установить зависимости с помощью uv:
```bash
uv sync
```

Это создаст виртуальное окружение в `.venv/` и установит все необходимые пакеты.

## Использование

### Запуск Python скриптов

Используйте `uv run` для выполнения Python скриптов в виртуальном окружении:

```bash
uv run python src/your_script.py
```

### Запуск Jupyter ноутбуков

Запустить Jupyter Lab или Jupyter Notebook:

```bash
uv run jupyter lab
# или
uv run jupyter notebook
```

### Запуск тестов

Выполнить тесты с помощью pytest:

```bash
uv run pytest
```

### Форматирование и линтинг кода

Форматировать код с помощью ruff:

```bash
uv run ruff format .
```

Проверить код с помощью ruff:

```bash
uv run ruff check .
```
